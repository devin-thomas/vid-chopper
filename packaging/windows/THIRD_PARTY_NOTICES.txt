Third-party runtime notice for the Windows release zip
======================================================

This release bundle is assembled from the VidChopper repository and the Windows runtime files copied by Qt's deployment tooling.

Bundled with the release zip
----------------------------
- VidChopper application binaries from this repository
- Required Qt 6 runtime files deployed by windeployqt
- Microsoft Visual C++ runtime files deployed by windeployqt

Not bundled with the release zip
--------------------------------
- ffmpeg
- ffprobe
