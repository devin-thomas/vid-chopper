Windows 10/11 x64 Release

VidChopper

VidChopper turns a single source video into chapter clips with a focused Windows desktop workflow, precise timestamp editing, GPU-aware encoding defaults, and a portable GitHub release zip that does not require a local Qt build setup.

Download Windows Zip All Releases Repository

The release zip bundles VidChopper.exe, the required Qt runtime, and the VC++ runtime. ffmpeg and ffprobe still need to be installed separately or configured in-app.

Portable Install Path

Download the latest GitHub release, unzip it anywhere writable on a 64-bit Windows machine, and launch VidChopper.exe directly.

What It Does

Load a video, start from embedded chapters or a six-part starter layout, edit names and boundaries, and export each chapter through ffmpeg.

What Still Needs Setup

The release zip does not ship with ffmpeg or ffprobe. Keep them on PATH or point VidChopper at custom locations in Advanced Settings.

Encoding Defaults

VidChopper prefers HEVC NVENC when both an NVIDIA GPU and hevc_nvenc are available. Otherwise it falls back to x264 for predictable compatibility.

Release Highlights

  • Portable Windows x64 zip attached to each published GitHub release
  • Qt runtime and VC++ runtime bundled beside VidChopper.exe
  • Qt 6 Widgets interface with a dark-only workflow
  • Pure C++ core for chapter planning and command generation
  • GitHub Actions CI for the core path, GUI build, and release packaging

Important Constraints

  • Windows is the primary target in the current repository state
  • Source builds still require a Qt 6 SDK for the full GUI target
  • The portable release still depends on external ffmpeg and ffprobe binaries
  • Exports are sequential to keep validation and progress tracking clear
  • Qt and FFmpeg licensing obligations remain separate from the repo license
